Un MVP (Minimum Viable Product), c'est la version la plus simple de ton produit qui permet de le tester avec de vrais utilisateurs. Pas une maquette, pas une démo — un vrai produit fonctionnel, avec authentification, base de données et toutes les fonctionnalités essentielles. Juste sans les features secondaires qu'on construira après validation.
SaaS MVP (999€) : authentification utilisateur, dashboard, base de données, déploiement sur ton domaine, 1 fonctionnalité principale.
